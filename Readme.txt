pip install requirements.txt

#Description💪🏻:
The proposed AI gym coach aims to provide personalized guidance to users during their workout routines, ensuring correct posture and minimizing the risk of injuries.

#Python Implementation🏋🏼‍:
=>Supported Exercise types
1- Pushup
2- Squat
3- Plank

#Steps of The Run Code💪🏻:
1- First, record the exercise you want to perform analytics on, and save it in a suitable path.
2- Run the Gym.py file
3- Select the type of exercise you want to perform.
4- Select the video path on which the analysis will be performed.
